Introduction

This tutorial will show you on hand of a real world example how you design, setup and implement an application with OOS. You will learn

  • how to design and create your classes
  • how to create object relations
  • how to setup the object store
  • how to setup a database connection
  • how to deal with database transactions

Goal

Goal of this project is to create a simple music database. With the resulting applicaction you'll be able to setup a little music database. The program will implement several commands which will work on the database.
