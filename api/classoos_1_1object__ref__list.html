Detailed Description

template<typename T>
class oos::object_ref_list< T >

An object list class for object_ref.

Template Parameters:
TThe concrete object type.

This is a specialisation of the object_list class. It stores object_ref in a list. The raw objects are inserted into the list and than subsequently inserted into the object_store. The reference link to the parent class is done automatically.

Creates an empty list.

A new object_ref_list is created. The list is part of the given parent object and therefor a reference to the parent object must be found inside the value type object with the given list_ref_name.
