Detailed Description

The Open Object Store namespace.

This is the namespace where all OOS related classes, function and code should be put into.

Enumeration Type Documentation

enum oos::t_convert_policy

Coversion policies. There are three policies:

  • strict: Input must be output type.
  • fitting: As long as the size of the types fits or it is a string.
  • weak: Conversion is done although data loss could take place.
Enumerator
convert_strict 

Strict policy for conversion

convert_fitting 

Fitting policy for conversion

convert_weak 

Weak policy for conversion

convert_strict_fitting 

Shortcut for strict and fitting policy combination

convert_fitting_weak 

Shortcut for fitting and weak policy combination

convert_all 

Shortcut for all policies combination
