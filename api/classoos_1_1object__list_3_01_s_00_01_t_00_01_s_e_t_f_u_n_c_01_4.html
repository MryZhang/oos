Detailed Description

template<class S, class T, void(T::object_type::*)(const object_ref< S > &) SETFUNC>
class oos::object_list< S, T, SETFUNC >

List class mapping items via relation table.

Template Parameters
SThe parent class type.
TThe item class type.
SETFUNCThe parent setter function.

This object list class uses no relation table. The items hold the information of its parent by themself.
