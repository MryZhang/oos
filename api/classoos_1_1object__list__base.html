oos::object_list_base::object_list_base ( object *  parent,
const std::string &  list_ref_name 
)

Creates an empty object_list_base object.

The constructor creates an empty object_list_base object which is part of the given parent object. This means that the parent object contains this list and each node object must have a reference field/parameter for the parent object set automatically on insert.

Parameters:
parentThe parent object of this list.
list_ref_nameThe name of the list reference parameter.
