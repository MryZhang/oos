Detailed Description

template<class T>
class oos::linked_object_ref_list_node< T >

Node type for linked lists with an object_ref as value.

Template Parameters:
TThe object type of the object_ref class.

This node class is specialization for the linked_object_list_node class for linked object lists. It contains only an object_ref and wraps it with direct accessors.
