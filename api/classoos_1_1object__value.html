Detailed Description

template<class T>
class oos::object_value< T >

Update and retrieve values of an object.

Template Parameters:
TThe type of the value.

The object_value class updates and retrieves a single value of an object in a generic way. Therefor the name of the value must be known. The name is given in the read_from and write_to methods of each and every object_store object.
