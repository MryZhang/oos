Detailed Description

Provides gemeric conversion functions.

This file provides generic conversion function. All builtin types, std::string and oos::varchar_base types are supported. User can select between three conversion policies:

  • strict: Input type must be output type.
  • fitting: As long as the size of the types fits or it is a string.
  • weak: Conversion is done although data loss could take place.
